Vergleiche und If-Verzweigungen

Erinnert ihr euch noch an das Beispiel ganz am Anfang, anhand der Funktion "if"? Diese werden wir nun behandeln. Mithilfe der If-Verzweigung könnt ihr überprüfen ob ein Ausdruck wahr oder falsch ist. Ein Beispiel hierfür wäre 5==4. Python überprüft hier ob 5 gleich 4 ist. Die wichtigsten Vergleiche sind:


Vergleiche

X==Y --> X ist gleich Y


X!=Y --> X ist ungleich Y


X<Y oder X>Y --> X ist kleiner bzw. grösser als Y


X<=Y oder X>=Y --> X ist kleiner gleich bzw. grösser gleich Y


Man kann ausserdem mehrere Vergleiche mit den "boolschen Operatoren" verketten.


Boolsche Operatoren

"Vergleich1" or "Vergleich2" -> Wahr, wenn einer der beiden Vergleiche wahr ist.


"Vergleich1" and "Vergleich2"-> Wahr, wenn beide Vergleiche wahr ist.


Not "Vergleich1" -> Ist Wahr falls Vergleich1 falsch ist und falsch falls Vergleich1 wahr ist.


Bei der If-Verzweigung wird, falls die Bedingung wahr ist, der jeweilige, vordefinierte Programmabschnitt ausgeführt. Sonst wird der vordefinierte Programmabschnitt nach dem else: ausgeführt. Man kann auch mehere If-Verzweigungen verschachteln mithilfe von elif. Diese kommt anstelle von else: und wird ausgeführt falls die erste Bedingung nicht zutrifft.


If-Verzweigung

if "Bedingung":


    "Code falls Bedingung zutrifft"


elif "Bedingung2":


    "Code falls Bedingung2 zutrifft"


else:


    "Code falls Bedingung2 nicht zutrifft"


Diese Bedingungen kann man ebenfalls als Variable speichern. Diese nehmen dann entweder den Wert "true" oder false" an. Diese Variablen entsprechen einem uns noch unbekannten Datentypen, nämlich dem Datentyp boolean. Da die If-verzweigung nur überprüft, ob der Wert des Codes "true" oder "false" ist (daher kommt auch der eigentliche Name für die "Bedingungen", nämlich "boolsche Ausdrücke"), könnt ihr eine "boolean" Variable anstelle der "Bedingung verwenden.


Datentyp: Boolean

Variable2=true


If "Variable2":


    "Dieser Code wird ausgeführt, da Variable 2 den Wert "true" hat"

Falls man einen Codeblock mehrmals ausführen will, kann man das bei Python mithilfe der "While"- und der "For"-Schleife umsetzen. "While" funktioniert ähnlich wie eine if-Verzweigung. Der angegebene Codeblock wird so oft ausgeführt, bis die Bedingung nicht mehr zutrifft. Bei der "While"-Schleife, sollte man allerdings aufpassen, dass die Schleife einen Endpunkt hat, da sie sonst unendlich lange weiterläuft.


"While"-Schleife

while "Bedingung":


If "Variable2":


    "Dieser Codeblock wird so lange wiederholt, bis die Bedingung nicht mehr zutrifft"


Die "For"-Schleife hingegen, weist einer angegebenen Variable (Bsp. X) nacheinander alle Zahlen eines gewissen Bereiches beziehungsweise einer gewissen Sequenz zu. Für jeden neuen Wert der Variable wird der Code in der "For"-Schleife einmal ausgeführt. Dabei kann man diesen Wert der Variable auch in der "For"-schleife verwenden.


"For-Schleife"

For i in range(x,y):


    "Dieser Code wird y-x mal ausgeführt und führt dabei den Codeblock für jede Zahl zwischen x und y einmal durch. Die Variable i nimmt dabei die entsprechende Zahl an."


For i in Liste1:


    "Dieser code wird so oft durchgeführt, wie es Elemente in Liste1 hat. I nimmt dabei, der Reihenfolge nach, den jeweiligen Wert an.


Wenn man die "For"- oder "While"-Schleife schon vorzeitig beenden möchte, kann man dies mit "break" bewerkstelligen.


Break

Break --> bewirkt eine vorzeitige Beendung der "For"- oder "While"-Schleife
